## 🔧 如果遇到问题

### 问题1：页面空白
**解决方法**：
1. 确认终端显示 `Running on http://127.0.0.1:8000`
2. 在浏览器中按 `F12` 打开开发者工具
3. 查看 Console 标签的错误信息
4. 查看 Network 标签确认 API 请求

### 问题2：端口被占用
**错误信息**：`Address already in use`

**解决方法**：
```bash
# 停止所有 Python 进程
pkill -f "python.*app"

# 等待 1 秒
sleep 1

# 重新启动
python3 app.py
```

### 问题3：数据加载失败
**可能原因**：
1. API 连接问题
2. 网络问题

**检查方法**：
在终端中查看日志输出，应该看到：
```
API Response Status: 200
Found 8 properties
```
